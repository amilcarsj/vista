General Information

Uploading Trajectories

When uploading trajectories it is important to remember:

  1. The current accepted format for trajectories is csv. Supported delimiters are (comma, semi-colon, tab, and pipe)
  2. If uploading multiple files, ensure that all csv files have the same column structure.
  3. If there are multiple trajectories in one file, specify the ID field to split into multiple trajectories based on that field.
  4. Latitude and Longitude values should be should be in WGS84 format
  5. The time field should in date time format.
  6. After submitting on the last screen you will be able to see your trajectories below.

Uploading POIs and ROIs

When uploading POIs and ROIs it is important to remember:

  1. The current accepted format is geoJSON FeatureCollection's (Even if you are only uploading one feature)
  2. POIs acceptable types LineStrings, MultiLineStrings, Points, MultiPoints.
  3. ROIs acceptable types are Polygons and MultiPolygons
  4. After uploading the layer please provide a name in the right column.

Creating Labels and Inviting Users

When Creating Labels and Inviting Users it is important to remember:

  1. Labels should be created using semi-colons(;) to separate the different labels.
  2. User's should be identified by their email and separated using semi-colons(;)
  3. All user emails should already be registered in the system.

Starting a Tagging Session

When starting a tagging session it is important to remember:

  1. You must be invited to at least one session, otherwise the dropdown menu will be empty.
  2. Select the tagging session you wish to open, and click Select Session.
  3. The tagging session will open and you should see something similar to the picture on the right.
  4. There are 3 controls under the zoom control in the top left. They will be explained in more detail below
  5. The legend in the bottom right of the map indicates the colour of the trajectory in correspondence to the value of its point feature.
  6. The arrow in the top right is to go to the next trajectory. To be able to proceed you must finish segmenting the current trajectory
  7. The Display Scatter Plot will toggle a scatter plot where you can select a point feature for the X axis and for the Y axis.
  8. The dropdown with the Point Feature label will allow you toggle what point feature you are looking at in the line chart below.

Map Controls

The tools below are meant to improve the segmentation process

  1. The layer control (shown first) is used to toggle POI and ROI layers using the checkbox. The first dropdown is used for changing the line density of the layer. The second drop down the line opacity. The third dropdown is the fill opacity. Finally, the color picker allows you to change the color of the layer on the map.
  2. The play control (shown second) is used to animate the trajectory over time. You can specify the number of seconds in the textbox then press play.
  3. The segmentation control (shown third) is used to create the segmentation. Clicking one of the circular + buttons will place a marker on the map. The marker is draggable and will bind to the nearest point of the trajectory.

Review Session

When reviewing a session it is important to remember

  1. You must be the tagging session manager of at least one session to be able to see a session summary
  2. You will select the session you want to review from the dropdown menu and press select
  3. Then the window on the right should be shown with point feature averages on the left, and segment feature averages on the right. Broken down by user.
